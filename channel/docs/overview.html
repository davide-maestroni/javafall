Parallel programming on the go.

Overview

Channel utility classes based on the jroutine-core library.

Through the module classes it is possible to combine input and output channels in several ways, so to enable complex concatenation of routines.

Main concepts

  • Flow

    Flow data allow for effective transmission of multiple streams through a single channel. Each object is linked to a specific ID so that is possible to discriminate between data coming from logically separated flows.

Main classes

  • Channels

    This utility class exposes several methods to combine and transform channels.

  • ByteChannel

    A byte channel allows for efficient transmission of binary data through a routine channel.

Dependencies

Below a full list of the included modules:

  • jroutine-core — core routine and channel classes

Usage examples

Example 1: blending the output of two channels. 
    
      final Channel<?, Object> channel = Channels.blend(channel1, channel2).buildChannel();
Example 2: concatenating the output of two channels. 
    
      final Channel<?, Object> channel = Channels.concat(channel1, channel2).buildChannel();
Example 3: joining the output of two channels. 
    
      final Channel<?, List<Object>> channel = Channels.join(channel1, channel2).buildChannel();
