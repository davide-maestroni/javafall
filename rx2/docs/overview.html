Parallel programming on the go.

Overview

Integration of RxJava2 based on the jroutine-core library.

This module provides classes integrating routines with the RxJava2 library.

Main classes

  • JRoutineFlowable

    Utility class mapping Flowable and channel instances.

Dependencies

Below a full list of the included modules:

  • jroutine-core — core routine and channel classes

Usage examples

The code below shows how it is possible create an Flowable from a channel, and the contrary as well.

Example 1: Flowable from channel. 
    
        JRoutineFlowable.create(myChannel).buildFlowable().subscribe(getConsumer());
Example 2: channel from Flowable. 
    
        JRoutineFlowable.with(myFlowable).buildChannel().bind(getConsumer());
