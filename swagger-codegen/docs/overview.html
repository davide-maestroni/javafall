Overview

Swagger generator based on the jroutine-retrofit library.

This module provides classes generating client code employing routines to perform network requests.

Main classes

  • JRoutineCodegen

    The Swagger generator.

Dependencies

Below a full list of the included modules:

  • jroutine-channel — utility classes extending channel functionalities
  • jroutine-core — core routine and channel classes
  • jroutine-function — utility classes enabling function-based routines
  • jroutine-object — routines wrapping object and class methods
  • jroutine-retrofit — Retrofit integration
  • jroutine-stream — stream-like routine concatenation

Usage examples

The code below shows how it is possible to generate, from a Swagger specification, a client calling the GitHub repo APIs.

Example 1: via code generation. 
    
        public class Main {

            public static void main(final String[] args) {
                final RepoApi service = new RepoApiClient().createService(RepoApi.class);
                service.getReposBuilder(args[0])
                       .map(Operators.<Repo>unfold())
                       .close()
                       .bind(onOutput(System.out::println))
                       .after(seconds(3))
                       .getComplete();
                System.exit(0);
            }
        }
