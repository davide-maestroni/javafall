Overview

Swagger code generator based on the jroutine-retrofit library.

This module provides classes generating client code employing routines to perform network requests.
The generator adds a custom "jroutine" language and additional command line options to enable support for Android classes. Run the generator with "config-help -l jroutine" for a complete list.

Main classes

  • JRoutineCodegen

    The Swagger code generator.
    The generated code will depend on the jroutine-retrofit library.

Usage examples

The code below shows how it is possible to generate, from a Swagger specification, a client calling the GitHub repo APIs.

Example 1: via default factory. 
    
        public class Main {

            public static void main(final String[] args) {
                final RepoApi service = new RepoApiClient().createService(RepoApi.class);
                service.getReposBuilder(args[0])
                       .map(Operators.<Repo>unfold())
                       .invoke()
                       .consume(onOutput(System.out::println))
                       .close()
                       .in(seconds(3))
                       .getComplete();
                System.exit(0);
            }
        }
Example 2: via Android Service. 
    
        public class MainActivity extends Activity {

            private static final String USER_NAME = "...";

            private static Consumer<Repo> addRepoTo(final ArrayAdapter adapter) {
                return repo -> {
                    adapter.add(repo);
                };
            }

            private static Consumer<RoutineException> showToast(final Activity activity) {
                return error -> {
                    final Throwable cause = error.getCause();
                    Toast.makeText(activity, (cause != null) ? cause.getMessage() :
                            "Cannot load repositories", Toast.LENGTH_LONG).show();
                };
            }

            @Override
            protected void onCreate(final Bundle savedInstanceState) {
                super.onCreate(savedInstanceState);
                setContentView(R.layout.main_layout);
                final ArrayAdapter adapter = new ArrayAdapter<>(this, android.R.layout.simple_list_item_1);
                final ListView repoList = (ListView) findViewById(R.id.repo_list);
                if (repoList != null) {
                    repoList.setAdapter(adapter);
                }

                final ServiceAdapterFactory adapterFactory =
                        ServiceAdapterFactory.on(serviceFrom(this)).buildFactory();
                final RepoApi service = new RepoApiClient().withAdapterFactory(adapterFactory)
                                                           .createService(RepoApi.class);
                service.getReposBuilder(USER_NAME)
                       .map(Operators.<Repo>unfold())
                       .invoke()
                       .consume(onOutput(addRepoTo(adapter)).andOnError(showToast(this)))
                       .close();
            }
        }
