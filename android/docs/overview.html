Parallel programming on the go.

Overview

Android platform customization of the jroutine-lib library.

The module collects all the main features by including the base modules and exposing them trough a single entry point class.
It is anyway possible to include just the needed sub-modules as dependencies.

Main classes

  • JRoutineAndroid

    Utility class exposing the JRoutine Android modules functionalities.

  • JRoutineAndroidCompat

    Utility class exposing the JRoutine Android modules functionalities. This class is specifically meant to employ the Android Support library for retro-compatibility.

Dependencies

Below a full list of the included modules:

  • jroutine-androidchannel — utility classes extending channel functionalities
  • jroutine-androidcore — core routine and channel classes
  • jroutine-androidmethod — routine implementation through a class method
  • jroutine-androidobject — routine wrapping object and class methods
  • jroutine-androidproxy — code generation of routines wrapping object and class methods
  • jroutine-androidstream — stream-like routine concatenation
