Parallel programming on the go.

Overview

Android customization of channel utility classes based on the jroutine-channel library.

Through the module classes it is possible to combine input and output channels in several ways, so to enable complex concatenation of routines.
Specific Android classes are employed to store maps of channels and to make data parcelable.

Main concepts

  • ParcelableSelectable

    ParcelableSelectable data allow for effective transmission of sub-flows of data through a single channel connected to invocations running in dedicated Services. Each selectable is linked to a specific index so that is possible to discriminate between data coming from logically separated flows.

Main classes

  • AndroidChannels

    This utility class exposes several methods to combine and transform channels.

  • SparseChannels

    This utility class exposes several methods to combine and transform channels supporting Android sparse arrays.

  • SparseChannelsCompat

    This utility class exposes several methods to combine and transform channels supporting Android sparse arrays.
    This class is specifically meant to employ the Android Support library for retro-compatibility.

  • ParcelableByteChannel

    A parcelable byte channel allows for efficient transmission of binary data through a Service routine channel.

Dependencies

Below a full list of the included modules:

  • jroutine-android-core — core routine and channel classes
  • jroutine-channel — utility classes extending channel functionalities
  • jroutine-core — core routine and channel classes

Usage examples

Example 1: blending the output of two channels. 
    
      final Channel<?, Object> channel = AndroidChannels.blend(channel1, channel2).buildChannels();
Example 2: concatenating the output of two channels. 
    
      final Channel<?, Object> channel = AndroidChannels.concat(channel1, channel2).buildChannels();
Example 3: joining the output of two channels. 
    
      final Channel<?, List<Object>> channel = AndroidChannels.join(channel1, channel2).buildChannels();
