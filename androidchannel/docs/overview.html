Parallel programming on the go.

Android customization of channel utility classes based on the jroutine-channel library.
Note that SparseChannels and SparseChannelsCompat utility classes, additionally supporting sparse collections, are also available with the same features as AndroidChannels class.

Usage examples

Example 1: blending the output of two channels. 
    
      final OutputChannel<Object> channel = AndroidChannels.blend(channel1, channel2).buildChannels();
Example 2: concatenating the output of two channels. 
    
      final OutputChannel<Object> channel = AndroidChannels.concat(channel1, channel2).buildChannels();
Example 3: joining the output of two channels. 
    
      final OutputChannel<List<Object>> channel = AndroidChannels.join(channel1, channel2).buildChannels();
