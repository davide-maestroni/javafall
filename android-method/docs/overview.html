Parallel programming on the go.

Overview

Routine method implementation based on the jroutine-android-core and jroutine-android-object libraries.

This module provides classes enabling a routine implementation through simple methods.
The classes allow asynchronous invocations inside dedicated Android Services and Loaders.

Main concepts

  • ServiceRoutineMethod

    Base class providing a simple way to implement a Service routine as an object method. The method will receive input through InputChannel instances and produce outputs through OutputChannel instances passed as parameters.

  • LoaderRoutineMethod

    Base class providing a simple way to implement a Loader routine as an object method. The method will receive input through InputChannel instances and produce outputs through OutputChannel instances passed as parameters.

  • LoaderRoutineMethod

    Base class providing a simple way to implement a Loader routine as an object method. The method will receive input through InputChannel instances and produce outputs through OutputChannel instances passed as parameters. This class is specifically meant to employ the Android Support library for retro-compatibility.

Dependencies

Below a full list of the included modules:

  • jroutine-android-channel — utility classes extending channel functionalities
  • jroutine-android-core — core routine and channel classes
  • jroutine-android-object — routine wrapping object and class methods
  • jroutine-channel — utility classes extending channel functionalities
  • jroutine-core — core routine and channel classes
  • jroutine-method — routine implementation through a class method
  • jroutine-object — routines wrapping object and class methods

Usage examples

The code below shows how it is possible to create a routine method printing different outputs based on their types..

Example 1: via Service routine method. 
    
        final InputChannel<Integer> inputInts = RoutineMethod.inputChannel();
        final InputChannel<String> inputStrings = RoutineMethod.inputChannel();
        final OutputChannel<String> outputChannel = RoutineMethod.outputChannel();
        new ServiceRoutineMethod(serviceFrom(activity)) {

            void print(final InputChannel<Integer> inputInts,
                    final InputChannel<String> inputStrings,
                    final OutputChannel<String> output) {
                final InputChannel<?> inputChannel = switchInput();
                if (inputChannel.hasNext()) {
                    if (inputChannel == inputInts) {
                        output.pass("Number: " + inputChannel.next());
                    } else if (inputChannel == inputStrings) {
                        output.pass("String: " + inputChannel.next());
                    }
                }
            }
        }.call(inputInts, inputStrings, outputChannel);
        outputChannel.bind(new TemplateChannelConsumer<String>() {

            @Override
            public void onOutput(final String out) {
                System.out.println(out);
            }
        });
