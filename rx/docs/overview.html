Parallel programming on the go.

Overview

Integration of RxJava based on the jroutine-core library.

This module provides classes integrating routines with the RxJava library.

Main classes

  • JRoutineRx

    Utility class creating an Observable from a channel instance.

Dependencies

Below a full list of the included modules:

  • jroutine-core — core routine and channel classes

Usage examples

The code below shows how it is possible create an Observable from a channel, and the contrary as well.

Example 1: Observable from channel. 
    
        JRoutineRx.observableFrom(myChannel).subscribe(getAction());
Example 2: channel from Observable. 
    
        JRoutineRx.with(myObservable).buildChannel().bind(getConsumer());
